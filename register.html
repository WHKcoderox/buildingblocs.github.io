Python fun task (adds to lucky bonus chance): A year is a leap year if it is divisible by 4 but not 100, or is divisible by 400. Write a Python program that prompts the user to enter a year and prints "Yes!" if the input year is a leap year, and "Leap year + <i>" (0 < i < 4) otherwise. Loop until -1 is entered.

Sample Run:
Enter year: 2016
Yes!
Enter year: 2017
Leap year + 1
Enter year: -1
Bye.

Email your Python program to buildingblocs@computing.sg. Happy coding!
